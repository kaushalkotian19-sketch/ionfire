# 🔥 IONfire

IONfire is a burn leaderboard platform for ION (BSC).

## 🚀 Features
- 🔥 Track ION burns
- 🏆 Top burners leaderboard
- 📊 Total burned stats

## ⚙️ Tech Stack
- Backend: Laravel
- Blockchain: BSC (BEP-20)
- API: BscScan

## 📌 MVP Plan
Phase 1:
- Total burned
- Top 10 burners

Phase 2:
- Wallet connect
- User stats

Phase 3:
- Profiles
- Badges
- Share system
